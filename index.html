MindTrack – Plataforma de Pruebas Cognitivas (App + Backend + Arduino)

Tecnologías: Android (Java), Node.js, Express (secure-api), PostgreSQL (Neon), Firebase Auth, Render, Arduino, Bluetooth HC-05.

Desarrollo FullStack de MindTrack es una plataforma diseñada para apoyar la aplicación de pruebas cognitivas en niños, integrando una app móvil Android, un backend seguro y sensores físicos conectados a Arduino.

  • Desarrollo FullStack de app Android para crear sesiones, registrar participantes, ejecutar pruebas y visualizar resultados básicos.
  • Integración de Arduino UNO + Bluetooth HC-05 y sensor de ritmo cardiaco para capturar datos fisiológicos durante las sesiones.
  • Diseño e implementación de una API REST segura (secure-api) con Node.js + Express, conectada a PostgreSQL (Neon) y desplegada en Render.
  • Integración con Firebase Auth para validar ID tokens en backend y permitir que solo usuarios autenticados creen y consulten sesiones.
  • Exposición de la secure-api para que una página web del proyecto consuma los datos; apoyo en la integración y pruebas de lectura según rol (usuario/administrador).
  • Elaboración de un checklist de testing (app, IoT, backend), documentación de errores y acciones correctivas antes de la presentación del proyecto.

Sistema de Punto de Venta para Barbería

Tecnologías: Android Studio(Java), Firebase Auth, Firebase Realtime Database.

Desarrollo FullStack de sistema tipo punto de venta desarrollado durante mi estadía en 6.º cuatrimestre para resolver un problema real de gestión en mi propia barbería, sustituyendo el registro manual de ventas por un sistema digital centralizado.

  • Implementación de autenticación de usuarios y administrador con Firebase Auth, controlando el acceso a funciones sensibles según el rol.
  • Desarrollo de un panel de venta tipo POS que permite seleccionar servicios y/o productos, capturar el monto recibido y calcular automáticamente el cambio a devolver al cliente.
  • Carga y gestión de servicios y productos, incluyendo manejo de stock y actualización de existencias con cada venta registrada.
  • Registro de cada transacción en Firebase Realtime Database, almacenando usuario, fecha, hora e información detallada de la venta para su consulta posterior.
  • Diseño de un panel administrativo para visualizar y sumar ventas por usuario y el total global en un rango de tiempo, facilitando el cierre de caja y el control del negocio.
  • Mejora del control operativo al reducir errores manuales en el registro de ventas y disponer de la información y el inventario concentrados en un solo sistema en la nube y en tiempo real.

Dashboard Web para Gastos Personales

Tecnologías: HTML, CSS, JavaScript, Firebase Auth, Firebase Realtime Database.

Desarrolo FullStack de aplicación web desarrollada para llevar el control de gastos e ingresos personales en tiempo real, con autenticación de usuarios, organización por categorías y un dashboard con estadísticas visuales.

  • Implementación de registro e inicio de sesión con Firebase Authentication, permitiendo que cada usuario tenga su propia información aislada y segura.
  • Diseño de una estructura en Firebase Realtime Database basada en usuarios/{uid}/categorias y usuarios/{uid}/movimientos, separando gastos e ingresos por tipo y categoría.
  • Funcionalidad para registrar movimientos (monto, fecha, categoría, tipo gasto/ingreso) y actualizar automáticamente los totales en el dashboard.
  • Dashboard interactivo con estadísticas y visualización gráfica del comportamiento financiero (totales por categoría, resumen de gastos vs ingresos, etc.).
  • Implementación de filtros en el frontend (por fecha, categoría y tipo) para analizar la información sin recargar datos desde la base cada vez.
  • Enfoque en una arquitectura limpia con separación en capas (data / services / presentation), facilitando el mantenimiento y la futura expansión del proyecto.

Maleta Inteligente con App Móvil

Tecnologías: Android Studio(Java), Firebase, Arduino, Bluetooth, GPS.

Proyecto IoT donde desarrollé una aplicación móvil Android que se conecta a un sistema basado en Arduino con módulo GPS y comunicación Bluetooth, permitiendo consultar la ubicación de una maleta en tiempo real.

  • Desarrollo de la app móvil en Android Studio, encargada de conectarse al dispositivo Arduino, recibir las coordenadas GPS y mostrarlas al usuario.
  • Implementación de inicio de sesión con Firebase para identificar al usuario y vincular la maleta a una cuenta específica.
  • Configuración de la comunicación Bluetooth entre la app y Arduino, manejando el emparejamiento y el intercambio de datos (latitud/longitud).
  • Recepción y uso de la ubicación GPS enviada por Arduino, permitiendo al usuario consultar la posición de la maleta directamente desde la app.
  • Diseño del flujo de consulta para que el usuario pueda ver la información de ubicación de forma clara (coordenadas y/o referencia de ubicación), mejorando el control y seguridad sobre la maleta.
  • Proyecto orientado a integrar móvil + hardware + nube, reforzando conocimientos de comunicación inalámbrica (Bluetooth) y uso de sensores (GPS) en un caso de uso real.
